### 接口(interface)
接口是一种类型
使用值接收者与指针接受者实现接口的区别
-   使用值接收者实现接口,结构体类型和结构体指针类型的变量都能存
-   指针接受者实现接口只能存结构体指针类型的变量
 
### 包(package)
-   import导入语句通常放在文件开头宝声明语句的下面
-   导入的包名需要用双引号包裹起来
-   包名称是从$GOPATH/src/后开始计算的,使用/进行路径分割
-   Go语言禁止循环导入包
